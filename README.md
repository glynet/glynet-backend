# Glynet Backend

API developed with Express and Prisma

## Installation

Glynet requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and Looplens and start the server.

---

Development
```sh
cd glynet-backend
yarn
yarn dev
```

Build & Run Production
```sh
cd glynet-backend
yarn
yarn build
yarn start
```
